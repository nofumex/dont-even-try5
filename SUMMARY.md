## 📝 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Добавить категорию "Конфеты"

1. /admin → 🗂 Manage catalog → 📦 Manage products → ➕ Add product category
2. Введите: Конфеты
3. Введите иконку: 🍬
4. ✅ Категория добавлена!

### Пример 2: Добавить товары в категорию

1. /admin → 🗂 Manage catalog → 📦 Manage products → ✏️ Edit category items
2. Выберите: 🍬 Конфеты
3. Нажмите: ➕ Add item
4. Введите: Шоколадные | candies_chocolate | 10
5. ✅ Товар добавлен!

### Пример 3: Загрузить файлы товаров

Отправьте файл .txt с именем содержащим папку:
- Файл: candies_chocolate_product1.txt
- Автоматически попадет в: data/candies_chocolate/

### Пример 4: Переключить подписку

1. /admin
2. Нажмите кнопку: 🔒 Subscription ON (или 🔓 Subscription OFF)
3. Статус обновляется в реальном времени
