# 📝 Примеры использования новых функций

## Пример 1: Создание категории "Конфеты" с подкатегориями

### Шаг 1: Добавить категорию

```
Админ-панель → 🗂 Manage catalog → 📦 Manage products → ➕ Add product category

Введите название: Конфеты
Введите иконку: 🍬
```

### Шаг 2: Добавить подкатегории

```
Админ-панель → 🗂 Manage catalog → 📦 Manage products → ✏️ Edit category items → 🍬 Конфеты → ➕ Add item

Вариант 1:
Шоколадные | candies_chocolate | 10

Вариант 2:
Конфеты без сахара | candies_sugarfree | 12 | 🍭

Вариант 3:
Мармелад | candies_marmelade | 8
```

### Шаг 3: Загрузить файлы товаров

Создайте файлы `.txt` с контентом товаров и отправьте в бот:
- Файл: `candies_chocolate_account1.txt` → попадет в `candies_chocolate`
- Файл: `candies_sugarfree_account1.txt` → попадет в `candies_sugarfree`
- Файл: `candies_marmelade_account1.txt` → попадет в `candies_marmelade`

## Пример 2: Управление подпиской

### Включить требование подписки
```
Админ-панель → 🔒 Subscription ON/OFF

Если нажать когда выключена:
→ Подписка включается
→ Новые пользователи должны подписаться на канал

Status: 🔒 Subscription ON
```

### Отключить требование подписки
```
Админ-панель → 🔒 Subscription ON/OFF

Если нажать когда включена:
→ Подписка отключается
→ Пользователи могут использовать бот без подписки

Status: 🔓 Subscription OFF
```

## Поток использования для пользователей

```
Пользователь запускает бота: /start
    ↓
Проверка подписки (если требуется):
    - Если НЕ подписан → Предложение подписаться
    - Если подписан → Переход в главное меню
    ↓
Главное меню:
    - 🛍️ Products
    - 📦 Stock
    - 👤 Profile
    ↓
При нажатии 🛍️ Products:
    - Видит все категории (Accounts, Proxies, Candies, ...)
    ↓
Выбор категории "Конфеты":
    - Видит товары (Шоколадные, Без сахара, Мармелад)
    ↓
Выбор товара "Шоколадные":
    - Видит кнопку "Buy | 10$"
    - Выбирает количество (1-5)
    ↓
Покупка:
    - Система проверяет баланс
    - Берет файлы из data/candies_chocolate/
    - Отправляет файлы пользователю
    - Удаляет файлы из папки
    - Снимает деньги со счета
```

## Администраторские команды (быстрая справка)

| Команда | Результат |
|---------|-----------|
| `/admin` | Открыть админ-панель |
| `🔒 Subscription ON/OFF` | Переключить требование подписки |
| `🗂 Manage catalog` | Управление каталогом |
| `📦 Manage products` | Управление динамическими категориями |
| `➕ Add product category` | Добавить новую категорию |
| `🗑 Remove product category` | Удалить категорию |
| `✏️ Edit category items` | Управлять товарами в категории |
| `➕ Add item` | Добавить товар в категорию |
| `💲 Change price` | Изменить цену товара |
| `🗑 Remove item` | Удалить товар |

## Совместимость

✅ **Полная поддержка обратной совместимости:**
- Старые категории (Accounts, Proxies) продолжают работать
- Новые динамические категории работают параллельно
- Пользователи видят все категории в одном меню
- Загрузка файлов работает для всех категорий

## Файлы конфигурации

### catalog.json
- Хранит все категории (старые и новые)
- Структура автоматически обновляется при первом запуске
- Вы можете редактировать вручную, но рекомендуется использовать админ-панель

### settings.json
- Хранит настройки бота (подписка, и т.д.)
- Создается автоматически при первом использовании
- Содержит: `{"require_subscription": true}`

### config.py
- Загружает переменные из `.env`
- `REQUIRE_SUBSCRIPTION` можно установить через `.env` (опционально)
